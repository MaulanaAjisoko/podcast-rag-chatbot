Chatbot berbasis RAG untuk bertanya tentang konten podcast dari file transcript YouTube.

# Fitur
-  Upload file transcript (PDF/TXT)
-  Chat interface untuk bertanya
-  Show source - lihat bagian transcript yang jadi referensi

# Cara Pakai
1. Cari file transcript podcast dari video di youtube yang terletak pada deskripsi video Show transcript
2. Setelah itu copy transcript dan jadikan file .txt atau .pdf
3. Setelah itu file bisa di upload dan di proses terlebih dahulu sebelum kita bisa menggunakan rag chatbot ini.
4. Kemudian anda bisa menanyakan hal-hal mengenai podcast tersebut.
5. Klik "Show Source" untuk melihat sumber jawaban
6. Saya sudah menyediakan 3 contoh podcast yang dapat digunakan.

# Teknologi

- LangChain
